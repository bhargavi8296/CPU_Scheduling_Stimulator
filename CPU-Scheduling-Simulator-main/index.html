Types of CPU Scheduling

FCFS (First Come First Serve)

  • Processes are executed in the order they arrive.
  • Non-preemptive.
  • Uses First-In-First-Out rule.
  • Can cause long waiting time if a long process comes first.

SJF (Shortest Job First)

  • SJF (Shortest Job First): Executes the process with the shortest burst time first.
  • Non-preemptive: Once a process starts, it runs till completion.
  • Efficient: Minimizes average waiting time.
  • Disadvantage: Can cause starvation for longer processes.

SRT (shortest Remaining Time)

  • SJF (Shortest Job First): Executes the process with the shortest burst time first.
  • Non-preemptive: Once a process starts, it runs till completion.
  • Efficient: Minimizes average waiting time.
  • Disadvantage: Can cause starvation for longer processes.
